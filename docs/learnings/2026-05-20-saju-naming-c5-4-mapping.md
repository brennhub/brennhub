## brennhub 학파 입장 (build-in-public 정신)

brennhub는 자원오행 214부수 매핑을 자체 구축하여 **"brennhub AI 학파(ai-default)"** 위치를 정립합니다. 정당성 근거:
- 팩트 기반 (환각 X) — 모든 매핑은 명시 source cross-reference로 검증 가능
- 字源 의미 분석 + 작명소 다수안 + 학술 abstract 인용
- 차이 부수 verbatim record (본 문서 아래)

향후 **Advanced 전통 학파 옵션** plug-in:
- 김기승 『자원오행 성명학』 (kim-kiseung) — Brenn 책 입수 시
- 이재승 2024 KCI 논문 (lee-jaeseung) — full text 접근 시
- 안태옥 ksname (ahn-taeok) — 부분 매핑 정리 후

**학파 우열 없음**. 사용자가 선호 학파 선택. brennhub 가치: 단일 정답 강제 X + 정직성 + 확장성.

## 2. 학파 비교

- **만장일치 (~120 부수)**: 자연·물질 부수 — 木·竹·艸·禾→목 / 火·日·赤→화 / 土·山·石·田→토 / 金·玉·刀·戈→금 / 水·氵·雨·川→수.
- **학파 차이 (~90 부수)**: 추상·신체·기호 부수. 학파마다 배속이 갈리거나 "불명" 처리. brennhub AI 학파는 null 회피 원칙상 1개로 강제 배속 (근거 §5).

## 3. 학파 plug-in 아키텍처

`lib/radical-ohaeng.ts`:
- `type School = "ai-default"` — 향후 union 확장.
- `RADICAL_OHAENG_AI_DEFAULT: Record<number, Ohaeng>` — 214 부수번호 → 오행.
- `SCHOOL_TABLES: Record<School, ...>` — 학파별 테이블 레지스트리.
- `getJaOhaeng(radicalNumber, school="ai-default")` — 학파 선택 조회.

## 4. 확장 path (전통 학파 추가)

1. `export const RADICAL_OHAENG_<SCHOOL_KEY>: Record<number, Ohaeng>` 추가 (예: `RADICAL_OHAENG_KIM_KISEUNG`).
2. `School` union에 학파 키 추가 (예: `| "kim-kiseung"`).
3. `SCHOOL_TABLES`에 매핑 등록.
4. UI Advanced 옵션 디자인 (BACKLOG 별도 task — "Advanced 전통 학파 옵션 추가").

예: 김기승 추가 → `RADICAL_OHAENG_KIM_KISEUNG` + `School` 에 `"kim-kiseung"` + `SCHOOL_TABLES["kim-kiseung"]` 등록.

## 5. 판단 근거 — ai-default 차이 부수 채택 로직

만장일치 부수 외, 추상·신체·기호 부수는 아래 cross-reference 우선순위로 1개 배속:

- **천간·지지 anchor** (최우선, 명리 정설): 乙→목 / 子·亥→수 / 辰·龍→토 / 辛·酉→금 / 午·馬→화. 부수 乙(5)·子(39)·辰(161)·辛(160)·酉(164)·龍(212)·馬(187)·鼠(208,쥐=子) 등.
- **오행-장부·감각 배속** (명리 정설): 心→화 / 目→목 계열이나 brennhub는 "보다=빛=화"로 目(109)→화 채택 / 耳→수(신장) / 自(코)·鼻→금(폐) / 舌→화(심) / 骨·齒→수(신장) / 肉(月,살)→토(비위) / 血→화(혈맥) / 毛·皮·革→금(폐).
- **색**: 玄·黑→수(흑) / 赤→화(적) / 白→금(백) / 黃→토(황) / 靑→목(청).
- **字源 사물**: 舟·片·爿·耒·聿→목(목제) / 瓦·缶·皿·臼→토(질그릇) / 刀·戈·矛·矢·斤·貝→금(금속·재물) / 冫→수(얼음) / 卜·爻→화(점).
- **다수안 채택** (학파 차이 큼): 手(64)→목, 口(30)→수, 月(74)→수, 行(144)·車(159)→화 — 안태옥·작명소 다수안.
- **잔여 추상 부수**: 字源 최근접 오행 (예: 几(안석)→목, 厂(기슭)→토, 殳(몽둥이)→금). 본 범주가 학파 차이 최대 — Advanced 학파 옵션으로 정정 가능.

## 7. 한계 — 학파 다양성 (한계 아님)

~90 추상·신체 부수의 배속은 **학파 간 자연스러운 차이**다 (이재승 2024도 "불명" 범주 인정). brennhub AI 학파는 팩트 기반 1개 안을 제시하되 단일 정답을 강제하지 않는다. 향후 Advanced 전통 학파(김기승/이재승/안태옥) plug-in으로 사용자가 선택 가능 — build-in-public 정합.
