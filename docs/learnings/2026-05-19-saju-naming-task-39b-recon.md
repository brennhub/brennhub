## 5. 메타 학습

1. **단순 뺄셈 ≠ 집합 차이** — `9,389 − 8,095 = 1,294`는 두 집합이 포함 관계일 때만 성립. 양쪽에 고유 원소가 있으면 실제 차집합을 계산해야 함. 이전 리포트가 이 함정에 빠짐.
2. **데이터 정규화 선행** — gov가 한자를 코드포인트 문자열로 저장 → 정규화 없이 비교하면 차집합이 2,218로 부풀려짐. 정규화 후 1,498.
3. **"cover 100%" 주장은 검산 필수** — 영어 fallback이 실제로 612/1,498만 가능. 수치 주장은 재현 가능한 스크립트로 검증.
4. **벽자의 실무 가치** — 법적으로 인명 사용 가능 ≠ 실제 사용. 교육용 한자 비율 / 강희자전 등재 / Unicode 블록이 실무 희귀도의 객관 지표.
